# Mail Survey Dashboard

This dashboard, built in Python using Dash, displays customer satisfaction results based on feedback collected through links embedded in email signatures. 

The visualizations provide a clear and dynamic overview of customer perceptions and trends.

## Installation

Clone the repository

```bash
git clone https://github.com/appox-ai/mail-survey-dashboard.git
```

Create a virtual environment

```bash
python -m venv .venv
```

Activate the virtual environment

```bash
source .venv/bin/activate
```

Install the dependencies

**Prerequisite**: dash requires Python v3.8 or later

```bash
pip install dash pandas
```

## Usage

```bash
python app.py
```
